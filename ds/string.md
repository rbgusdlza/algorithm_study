## 자주 사용하는 메서드

### 1. +=

```
string a, b;
a = "bob"; b = "joa";
a += b;
cout << a; // bobjoa
```

* 문자열과 문자열을 더할 때 이용한다.
* 해당 문자열을 앞에 더하고 싶을 때는 a = b + a 로 표현한다.

### 2. size()

```
string a = "bob";
cout << a.size(); // 3
```

* 문자열의 크기를 반환한다.
* O(1)의 시간복잡도를 가진다. (string class 내부 변수로 저장되어 있기 때문)

### 3. insert(위치, 문자열)

```
string a, b;
a = "joa"; b = "bob";
a.insert(0, b);
cout << a; // bobjoa
```

* 특정 위치에 주어진 문자열을 삽입한다.
* O(n)의 시간복잡도를 가진다. (기존 문자열 처음 또는 중간에 삽입될 경우,  해당 위치 이후의 모든 문자들을 이동시켜야 하기 때문)
